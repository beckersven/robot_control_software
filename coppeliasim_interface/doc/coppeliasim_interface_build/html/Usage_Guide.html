Usage Guide¶

To use the CoppeliaSim-Interface, one has to do following steps in correct order. In the beginning, no ROS or CoppeliaSim should be running. Be sure to always source required bash-files!

  1. Open 3 terminal windows
  2. In the first one, start the ROS-framework with roscore
  3. In the second, set the sim-time-parameter true through: rosparam set /use_sim_time true
  4. Also in this terminal, navigate via cd to CoppeliaSim and launch it: ./coppeliaSim.sh
  5. Wait until CoppeliaSim is launched, then open and configure the basic_scene.ttt
  6. In the third terminal, run: roslaunch coppeliasim_interface simulation_bringup.launch
  7. Wait until MoveIt’s GUI is fully loaded and perform basic planning-/execution-tasks to ensure functionality
